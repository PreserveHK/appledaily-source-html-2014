男生圍：直擊日本橫濱CP+　G1 X Mark II 採櫻　5cm近攝

四十多年來日本最強暴風雪剛完結，轉眼間，大家竟然可以拿起相機拍攝櫻花？那不是天氣變幻無常，而是橫濱一年一度的大型相機展──CP+正式舉行。綜觀各大品牌走勢，今年似乎是體積輕巧的半專業相機繼續雄霸半壁江山，正如型號Canon G1 X Mark II，真機場內首度現身，它更以色彩繽紛的櫻花，配上細緻紋理，供途人任拍任影，將拍攝變成採花活動，原來也是一種享受。

記者、攝影：陳國輝

今屆參與CP+（Camera & Photo Imaging Show）的品牌，除Canon和Nikon兩大巨頭外，當然少不了剛有新作發表的Sony、Olympus和Fujifilm，會場面積只有一層，但勝在人多熱鬧。更難得的是，今屆參展商還有一些「卧虎藏龍」，如近年轉攻數碼化的經典牌子Hasselblad、德國鏡頭廠商Zeiss，甚至Sigma也推出外形奇特的DP2。最大轉變是，傳統品牌不再是會場唯一看頭，如想進一步了解各大相機界勢力發展，記得留意今明兩天的報道。

進化至31點對焦

據日本科技網站爆料，原來Canon早於CP+前一日已經公佈G1 X Mark II的規格和售價，並於三月中旬發售，但講到正式落場體驗並試玩，就要親身入CP+才體驗得到。Canon展館面積頗大，分成展示舞台、數碼輕便相機、新概念相機、望遠鏡頭體驗等幾大區域，還有EOS學園講師討論會等。據稱，近期熱爆的藝人能年玲奈，首日亦有親自到場打氣，更播出其最新M2代言廣告，可惜當日無機會遇上。
講到Canon展館最當眼位，首選肯定是「數碼輕便相機」區域，從遠處望過去，只見在大光燈照射下，矗立着茂密如叢林的櫻花陣，襯托着一堆冷冰冰機器，竟有幾分大自然的味道，有趣在有很多途人竟在陣內「採花」！走近了，才發現原來有七至八部G1 X Mark II供人試拍，採花目的只得一個，就是要讓人感受新機的微距快感。

G1 X Mark II為處理上一代遺留下來的缺憾，特別在近攝功能下工夫，最明顯是微距模式，最近對焦距離將由以前的20cm大幅減少至5cm，見現場人士試拍，鏡頭幾乎與櫻花面貼面接觸，擺明主攻植物紋理大特寫！同時，對焦範圍亦比以往更闊，上一代G1 X只有9點，到今次已具備31點對焦系統，再加上於鎖定對焦下，最高連拍速度達到每秒5張，比起上一代只有3張，亦算一大進步。行入展館中間，還擺放着接近兩米高的G1 X Mark II名家作品，包括由日本攝影師吉村和敏和石橋睦美親自拍攝的懸崖建築和竹林美景，作品同樣表達到大自然的美態。

屏幕180度向上翻自拍

從數據來講，G1 X Mark II絕對稱不上頂級，像素只有1,280萬，焦距僅24-120mm的五倍光學變焦，甚至不能更換鏡頭，對玩開專業級的發燒友，的確不大吸引。
或許，改良才是今次新機的重點，除微距和連拍功能得到提升，記得上一代G1 X的設計，採用向左翻出式屏幕，並加入旋轉角度，雖然高拍低炒扭一扭屏幕即可，但問題是雙手同時舉機的話，操作會比較困難。為解決這缺憾，G1 X Mark II將屏幕改為可直接向上或向下翻，向上最多可調校至180度，剛好可對住自己玩自拍；至於身處人群當中，你亦可單手舉高相機，然後將屏幕向下翻，直接調校至看得舒服的角度，這次改良，可算是方便一班喜歡旅遊攝影或有工作需要的人士。

Profile

感光元件：1.5吋HS CMOS
像素：1,280萬
焦距：24-120mm（相當於35mm）
感光值：ISO100-12,800
微距拍攝：5cm
連拍：每秒3張（鎖定對焦為5張）
