1. 吳克儉形容「李飛普選」是「大躍進」，這樣，文革還會遠嗎？下一波，愛字頭紅衛兵長驅直進校園，支持公民抗命的學生和教授，被迫跪玻璃、剃陰陽頭！
2. 羅范認為學生罷課，「冇乜作用」。學生被支持鎮壓、平暴有理的差佬闖入校園，見人就亂棍暴打，學生大聲喊救命，打九九九，接線那個竟是曾偉雄！的確「冇乜作用」，打多兩棍！
3. 范徐說戴耀廷搵學生做先鋒，累死後生仔。捐過腎給女兒的范太，即刻扮成大媽款，拖住學生的……不是手，是後腿！阻截他們罷課，唔聽話麼？做他們手腳！
4. 狼凸叫年青人獨立思考。咦，這不就是叫港孩擁獨嗎！狼凸以個人身份教唆「港獨」，巧陰毒！
5. 十三學者與十八學者的方案被閘住，學者冇job可done，政改三人組同樣被閘住，唔使do。他們側側膊，糧照出，蛇住先。學生唔想被屈蛇，宿舍緊急落閘。指鹿為馬的黑暗時代來了，馬路見！
