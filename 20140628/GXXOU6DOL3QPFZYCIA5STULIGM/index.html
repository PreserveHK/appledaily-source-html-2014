巴黎膺最不歡迎旅客城市

法國巴黎人出名冷傲，省錢網站Coupon Code Pro最新調查，就選出巴黎（圖）為「最不歡迎旅客」的城巿。調查訪問了全球數千名旅客，多達13%覺得巴黎人不友善，其次是英國倫敦，有9%受訪者感覺不爽，連港人熱愛的旅遊點日本東京也排名第五。至於最歡迎旅客的城巿榜，則由美國紐約市爆冷居首，中國北京則屈居第二。
美國《赫芬頓郵報》
