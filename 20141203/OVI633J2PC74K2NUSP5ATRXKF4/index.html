與時間競賽！內地富豪愛駒
未來一個月非勝不可

上季最佳一哩馬「大運財」的同主馬，內地富豪潘蘇通名下，未曾在港上陣的高廐馬「大印銀紙」，繼上月18日在港初次試閘跑第二後，今日(2日)在沙田第三組1000米草閘中再配韋達，末段以凌厲後勁追過對手勝出。
「大印銀紙」出自Mizzen Mast，與「左右逢源」同父。在英國屬鴨都拿王子名下，曾四次上陣，獲兩冠一季，贏1400及1600米，雖然只勝出處女馬賽及普通讓賽，惟其同父同母之Jibboom十分厲害，曾勝出1400米泥地二級賽Raven Run Stakes、1600米泥地三級賽Buena Vist讓賽及1300米草地三級賽Monrovia讓賽，最出色一戰是泥地1400米一級賽聖蒙尼嘉讓賽跑獲亞軍。估計「大印銀紙」是泥、草皆宜的中距離好手，評73分必有利，但亦因為只評73分，要趕及爭分跑下月18日經典一哩賽一級賽絕不易，必須在下月12日左右公佈正選名單前贏馬，只剩下約一個月時間。
