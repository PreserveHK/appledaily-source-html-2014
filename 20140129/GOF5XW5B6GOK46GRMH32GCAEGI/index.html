沙膽虹手記：
Nielsen忽略了電子平台 - 沙膽虹

Nielsen Media Index剛公佈由2012年10月至2013年9月的12個月間收費和免費報紙的讀者人數調查結果，《蘋果日報》報紙版的讀者人數為941,000，排名第三，輸給《頭條日報》及《東方日報》；但若單以網站的閱讀率，《蘋果》則排行第一。由於Nielsen Media Index的網站閱讀率並不包括手機App，因此結果顯示的讀者人數只480,000，大大低估了《蘋果》電子版的總閱讀人數。就《蘋果》自己的數據顯示，網站點擊率只佔總數的不足三成，反而是iPhone和Android App的點擊最高，接近五成，來自iPad的點擊也近兩成，若然Nielsen Media Index的調查連App的也計算在內，《蘋果》電子平台讀者人數，按比例應達180多萬，是報紙的一倍。市場部同事表示會跟Nielsen商討，希望將App也加入調查，以更準確反映日益重要的電子平台閱讀率。
