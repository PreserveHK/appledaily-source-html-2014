拉布，必不可少 - 陳也

一，議員拉布，其他議員夾份拉茶，或者咖啡拉花，拉拉拉，來來來，全局同聲同氣搞美食會。
二，拉布，冇人會突擊考核那位民建聯口語狀元，英文係點叫法。他索性自創，拉布的英文，就是poopoo。第一個p是pull，第二個p是布的邪音。係呀，邪音，You dun think so？So what先！
三，拉布正在進行，除非被豆沙包擲中，否則議員可以繼續上網炒股，打機攞點數，睇韓劇，跟AV達人裸聊，神不知主席不覺。
四，議員還可以接力拉布，而不是被自由行厚多士阿嬸拉隊走進去拉大糞、吃榴槤，將議會MK和地鐵化，阿彌陀佛，是真正香港仔的小確幸。
五，中港罵戰和樓市都會升溫，拉布也有趨勢。升溫後，由拉布變揭布，議員衣着，布料越來越少，遲早一日，赤裸裸顯示效忠基本法，愛國愛港，一眼睇晒，沒有顛覆的空間。着住底褲宣誓就職，也算是奇觀，養眼指數，高過中環海濱那台隨時撞正高空降糞的摩天輪。買少啲衫，為堆填區減啲壓啦。將軍澳就快被逼上梁山，搞區獨！
