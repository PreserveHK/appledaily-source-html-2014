籲停用兩胰島素輸注器

【本報訊】英國藥物及健康產品監管局發出安全警示，指由Cellnovo生產的兩款胰島素輸注器，不銹鋼針頭可能會自行脫落。本港衞生署提醒市民停用該兩款輸注器，暫未接獲有人使用後有不良反應報告。
英國藥物及健康產品監管局發出的安全警示，指出Cellnovo生產的胰島素輸注器Infusion set 5mm stainless steel（型號：5mm-INF5）及Infusion set 8mm stainless steel（型號：8mm-INF5）出問題，病人更換新的胰島素匣子及移除該輸注器時，不銹鋼針頭可能會脫落，或需由醫護人員移除針頭。
衞生署表示，正與生產商了解該兩款輸注器有否分銷到香港，提醒市民停用該兩款產品。
