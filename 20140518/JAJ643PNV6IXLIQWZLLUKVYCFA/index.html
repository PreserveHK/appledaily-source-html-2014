自家手製狗仔粉

1. 燒熱鑊爆香乾葱，下蝦皮炒至金黃色作湯料。

2. 再起鑊，加入三碗水及100克粘米粉，拌勻煮至沒粉粒，熄火。

3. 倒入400克粘米粉及少許澄麵，趁熱搓至混合。可加適量熱水或粘米粉調節粉糰乾濕度。

4. 將麵糰分成小份，輾至約一厘米厚，切成約一厘米粗、七厘米長的條狀，可撒少許粘米粉以免麵條黏着。

5. 把麵條放在滾水滾10分鐘，加湯料，再煮約10分鐘，間中攪拌以免麵條黏在一起。熄火，焗最少半小時，再拌上配菜即可食用。

十八座狗仔粉煮法

1.將豬骨湯加入洋葱、蝦米仔、冬菇、豬油渣煮成湯底。

2. 加入已切好的麵條，先煲約一小時。

3. 煲好後蓋上煲蓋焗約半小時，變成稠身。

4. 最後加入葱花、豬油渣、菜脯即可享用。
