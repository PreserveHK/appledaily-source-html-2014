撼的士鏟行人路再撞貨車百萬電跑變「三文治」

撼的士鏟行人路再撞貨車
百萬電跑變「三文治」

【本報訊】宣傳公司東主昨午駕駛落地僅個多月、價值逾100萬元的Tesla Model S電動跑車，駛經九龍灣海濱道時與一輛的士相撞後，再撞向另一輛停泊於路邊的客貨車，跑車變「三文治」夾於兩車中間，的士更被攔腰撞至傾斜擱於橋躉，司機及兩名乘客一度被困，由消防救出送院，四人均無大礙。
記者：劉永明

昨午近2時，36歲姓李宣傳公司東主，駕駛紅色Tesla Model S（85kWh）電動跑車，沿海濱道往九龍城方向行駛。途至與啟興道交界時，姓賀（51歲）的士司機恰巧由德福花園接載一對姓吳（27歲）及姓陳（35歲）男女乘客，由相反方向的海濱道右轉入啟興道，跑車收掣不及，攔腰撞向的士左邊車身，兩車一同鏟上行人路，撞向觀塘繞道天橋的橋躉，跑車再撞及停泊路邊的一輛速遞公司客貨車。

乘客：使唔使揸咁快呀

意外後，的士被撞至傾斜擱於橋躉及鐵欄上，欄杆及路牌亦被撞毀，跑車則變「三文治」夾於兩車中間，賀及兩名男女乘客被困車內，需由消防協助救出，幸三人僅頸、腳受輕傷，李亦報稱頭暈不適，四人一併送院後並無大礙，姓蕭（39歲）客貨車司機則無受傷，在場協助調查，警方正追查車禍原因。
事後賀指自己駕駛的士31年，事發時根本避無可避：「一轉完個彎架車就好高速咁駛埋嚟，一見到嗰下就已經撞咗。」他又指兩名乘客亦嚇破膽高呼：「使唔使揸咁快呀！」

涉事的電動跑車，由姓李司機開設的宣傳公司持有，於本年9月底以102萬元登記買入，該型號車輛分為三級，涉事車輛屬中級，由0加速至100公里只需5.6秒，最高時速達225公里，續航距離達460公里，有指最高級別的續航距離更超過500公里，為電動車之最，而且啟德郵輪碼頭、灣仔及荃灣均有免費充電站，20分鐘便將電池充至半滿，深受富豪歡迎。
