賽後有古講：巴神轉死性獲讚賞 - 何西

利物浦在周日英超作客出戰熱刺，以3:0擊倒對手。新兵巴洛迪利首次披上紅軍戰衣成為比賽焦點，儘管他在今仗沒有入球，但表現仍廣受好評，領隊羅渣士賽後點名讚賞。

羅渣士：well done

以1,600萬鎊轉會費由AC米蘭轉投利物浦，巴洛迪利在周日比賽首次上陣，打破熱刺今季全勝強勢。巴神雖無入球，但表現賣力之餘又沒有發脾氣，於上半場受到對方守將艾歷迪亞的雙腳攔截，依然沒有發作。到61分鐘被調離場，在旁的領隊羅渣士亦不禁說一句：「做得好，小子（well done son）。」羅渣士賽後說：「巴洛迪利定必會對他未能在首仗建功感到失望，我相信今次是他生命中首次在防守角球時緊盯對手。他是意大利國腳，亦曾贏得歐聯，不過他加盟後對我說從來不會參與角球防守。我要求他按戰術去做，他在今仗亦有遵從，並曾以頭槌解圍；所以我認為，將他視為成年人看待，便會看見球場上最出色的巴洛迪利。」

周日賽前曾傳出巴神希望代替隊長謝拉特成為球隊頭號極刑劊子手，但比賽中最終仍是讓隊長操刀。謝拉特在該仗射入12碼，是他歷來為利物浦第43次極刑建功，打破昔日名將摩比的紀錄。羅渣士回應說：「我依然視謝拉特為頭號劊子手。」文：何西
