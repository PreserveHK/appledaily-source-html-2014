焦點故事：名師家品最平一折

名師家品與藝術品的共通點，除了要懂得欣賞，還要買得起。懂得欣賞但買不起，可以趁開倉入貨，低至一折買芬蘭設計師Eero Aarnio的Parabel Table，或者低於四折價錢買丹麥設計師Verner Panton的Panton Chair。

記者：伍志輝

攝影：黃偉傑

模特兒：Ivana（Mouton）

資料來源：Tint Int Ltd

其實Designer Chair可算是名師家品的代名詞，一張張透明膠製、線條利落、鮮艷奪目的Designer Chair，能夠為家居帶來一份濃烈的時尚感。當中不少經典之作多屬上世紀六七十年代的製品，部份甚至是五十年代的設計，時至今日仍然是前衞的標記，家具的線條色彩意念歷久不衰，正是經典Designer Chair的最大吸引力，亦受不同年代室內設計師鍾情。

吊椅設計玩味十足

以經典玻璃纖維椅Ball Chair為例，屬Eero Aarnio於1963年的設計，「房間中的房間」意念，只要坐進隔音功能極高的球形座椅中即能感受得到。Eero Aarnio於1968年設計的另一張玻璃纖維椅Pastil Chair，以及同年的透明Acrylic（聚酸甲酯，又稱亞加力膠）吊椅Bubble Chair，時至今日，依然玩味十足。
九十年代的作品如玻璃纖維製Parabel Table，如將藝術雕塑家具化，玻璃纖維製的巨大螺絲Screw Table，則是將童話巨人國的意念家品化。
這些歷年經典作品，正價由2,000多元至5萬多元不等，開倉價由一折至九折不等，視乎家品狀況而定，折扣越大，表面損耗可能越明顯。
Tint Int Ltd銷售部Andy表示，一些經典中的經典名師家品，折扣可能不會太大，如1960年的Panton Chair，堪稱史上首張塑膠製Designer Chair，依照人體功學設計的流線型一體化座椅，塑膠物料由早期的polyester（聚酯纖維）加玻璃纖維，到長青的polyurethane（聚氨酯）以及現代最普遍的polypropylene（聚丙烯），出廠價錢仍然每年有上漲空間，以低於四折的開倉價入手，可謂早買早享受。
其他還有跨界別設計鬼才法國Philippe Starck 的多款polycarbonate（聚碳酸酯）及polyethylene（聚乙烯）透明家品，同樣是有價有市的精品，由於保養得好沒有明顯花痕，一般折扣約為正價七折。
今個月底前可到北角海景大廈C座7樓參觀，開倉貨品近千件，包括傢俬、擺設及燈飾等。

設計界Lady GaGa　作品現代瑰麗

【名師介紹】
崇尚現代瑰麗感的設計，近年被紐約時報譽為「設計界Lady GaGa」的Marcel Wanders，可能正是你杯茶。1963年出生於荷蘭的Marcel Wanders，成長於經典設計師發光發熱的年代，1995年已經開設個人事務所，翌年以Knotted Chair 聞名於世，2001年與Casper Vissers成立家品公司Moooi，除了生產Marcel Wanders的瑰麗設計如閃亮幾何切割的Stone座椅，還有多位年輕設計師的產品，如瑞典三人女子設計組合Front的動物家品系列Pig Table，設計大膽而又充滿生活趣味性。去年開幕的灣仔精品酒店Mira Moon，設計總監正是Marcel Wanders，以中國神話故事「嫦娥奔月」為設計概念，演繹出一份富有現代優雅瑰麗感覺的中國韻味。

專門店選購　避免買假貨

【注意事項】
香港市面上大部份名師家品都來自意大利家具品牌，絕大部份家品仍然由意大利原廠製造，產品製作至檢測標準依然屬於世界最前列。透過專門店購買名師家品當然是最有保障的途徑，不同款式即使無現貨，專門店會代為向意大利總公司訂貨。
不少名師家品都有贋品，一般消費者單從表面看難以分辨真偽，Tint Int Ltd銷售部Andy提醒大家，辨別正牌貨的不二法門是看產品證書。
