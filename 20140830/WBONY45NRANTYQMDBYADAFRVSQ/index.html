周六休遊：Run and Fun　小跑團起動

早前一大夥人跑上寶雲道，以行動向強搶跑步徑的霸道工程說不，活動正是由跑團 Pow Bow策動。小跑團正在各自發功，改寫「跑步等於馬拉松」的觀念。不必定下半馬目標，一班人一齊捱，汗水也是甜的。走訪幾個好玩主題跑，我不跑步也想落場，跑！
記者：曾冠輝
攝影：劉永發、潘志恆、陳永威
部份圖片由受訪者提供

跑步去捍衞跑步徑
Pow Bow

剛剛成軍一年的Pow Bow，名字與「跑步」的粵音相近。五位核心成員中，沒一個是專業跑手，相反大多是新手落場，想跑，更想多人一齊跑，結果變成了社交活動。搞手之一的阿基說：「我們一群人關係千絲萬縷，A是B的中學同學，同時又是C的大學同學。」每次團跑都可能有新面孔，言談間才發現原來某某是mutual friend。彷彿因跑步而結緣於網上，又因跑步而集結於現實，重建實在的人際關係。
Pow Bow是個較關注社區的跑團。8月中，他們得悉紙業大王馮新聰入紙城規會，申請將於其寶雲道17號大宅附近的跑步徑改建成接駁大宅的車道，於是立即於網上號召跑友一齊跑上寶雲道，拿起螢光棒架成交叉，向破壞環境的工程說不。「寶雲道是我們非常、非常熱愛的一條緩跑徑，這項工程卻牽涉到享用寶雲道的公眾，還影響綠化地帶，我們希望政府三思。」這個跑團的活動未必很多人參與，但正是他們喜歡的規模，「曾參加些大跑團的活動，因為人多，結果跑友間的互動較少。反而人少少，剛好湊成一圍枱吃飯，可以與跑友們邊吃邊交流。」

團跑日：周四晚
宗旨：「每次跑步都是一場小旅行」
網址： http://www.facebook.com/powbownow

接力隊啤　瘋狂Beer Run
Mad˙One

某個周末晚上，另一夥人集結在中環10號碼頭，整裝待發等待狂熱的Beer Run。啤酒，看似與跑步毫不相干，只會傷肝，合起來竟然變得好玩有趣。「外國早有同類型活動，而且我們不是『劈酒』，只是淺嚐，反正很多人跑步後都會喝，何不將把兩者結合，一齊盡興？」曾志豐（豐教練）是跑界常聽到的名字，他是前港隊成員，亦是香港田徑總會一級教練。跑團名字中的「Mad」，即是「瘋」，與「豐」同音。人如其名，豐教練的跑團跑得瘋狂，每周兩天團跑，平路斜路統統不放過，還會做系統式體能訓練，夏天還會有泳池training。
當晚Beer Run四人一組，每人先跑400米，然後自隊一罐啤酒，再由隊友接力。是次召集了84人參加，加上跑團成員，共約百人齊集在碼頭外，是小跑團活動中較多人的一次。「坊間跑步賽事已夠多，我們不用再辦正規跑步活動，反而加入有趣主題，跑得更開心。」眼見如 The Color Run等國際大型主題跑巡迴世界各大城市，都入不到香港；Zombie Run千呼萬喚才通過政府重重難關，於10月在西九開跑，由民間小跑團各自舉辦的主題跑似乎較易成事。

團跑日：周二、五
宗旨：「輕輕鬆鬆一齊跑，開開心心一齊食」
網址： http://www.facebook.com/MAD.ONE.CREW

跑得嚒　不一定冇鼻哥
Run More跑得嚒

跑團「Run More 跑得麼」的中英名字看似牴觸，其實「跑得嚒，所以 Run More」吧。跑得麼的成員可謂是穿梭於各跑團活動的最常見捧場客，特別是Rachel，採訪Beer Run和舊鞋收活動都遇到她。「廣告人日夜顛倒，十個有九個都不做運動，放工和放假不外乎唱K和飲酒，不趁年輕運動一下很快『殘』。」經常帶着四塊腹肌遊走在各團跑的Rachel，曾經在意數字，今天總要比昨天跑更快，現在卻說：「不用快，最緊要好玩」，這是亦跑得麼的宗旨。

團跑日：周五晚
宗旨：「不用快，最緊要好玩」
網址： http://www.facebook.com/pages/BC-RUN-MORE/159775410884328

肯亞徐教練　送鞋到肯亞
Runfinity

蘇凱男是跑圈內另一星級教練，將徐濠縈練成「肯亞徐」的，就是他。由他領團的 Runfinity跑步之餘，希望多做一些有意義的事。最近他便聯同友好跑團Bring in Change舉行舊跑鞋收集活動，把收集到的跑鞋及跑衫送到肯亞。「香港跑手的資源實在太豐富，家中往往有十數對不同款式的跑鞋，以應付不同的練習場地，反觀肯亞的跑手練跑時可能有鞋也捨不得穿。」
跑手家中十數對跑鞋實屬等閒，還有從每個賽事中得到的跑衫也夠逼爆衣櫃，所以蘇教練另有想法，「很希望每年都可舉辦比賽，不派衫，只收衫。」他的目光不只在遙遠的肯亞，近如內地或本地的低收入家庭，他都希望能幫助、希望能帶着小朋友們一齊跑。

團跑日：周日早上
宗旨：「以跑步結伴，以跑步看世界」
網址： http://www.facebook.com/Runfinity

跑團推薦路線

Pow Bow
路線：沙田城門河
源禾路體育館→沿河畔向火炭方向或過橋到對岸→源禾路體育館
原因：他們曾探測PM 2.5有害懸浮粒子數值，發現此處比鬧市低得多。加上源禾路體育館企理，到禾輋陳根記大牌檔又方便。

Mad．One
路線：石硤尾
石硤尾公園體育館→南昌街上斜路→歌和老街→九龍塘→香港城市大學→石硤尾公園體育館
原因：有平路有斜路，可作多變化練習，跑畢可到南山邨車仔檔吃手打豬肉燒賣。

Runfinity
路線：南生圍
鳳琴街體育館→南生圍→鳳琴街體育館（沿路或乘橫水渡跑1公里回程）
原因：沿途不多車，可跑過濕地、河邊和可供人跑步的大草地，10和11月可能會看到候鳥。

跑得麼 Run More
路線：西九
九龍公園→西九海濱長廊→果欄→上海街→玉器街→九龍公園
原因：九龍公園室內運動場設有儲物櫃，更衣室可洗澡。途經果欄及上海街可欣賞戰前建築，晚跑可感受香港獨有果欄風情。

密密小跑啱新手

衞生防護中心出版的刊物中顯示，七成跑手於一年內均有下肢肌肉或骨骼損傷。註冊物理治療師陳詩薇指，跑步引致的問題主要集中腰、膝、小腿及足部，大部份屬勞損問題。她建議新手應以中強度、短時間（15至30分鐘）及高頻率（一周3至5次）訓練，然後慢慢加量。至於如何界定中強度訓練，可看跑步時能否作簡單對話，可以者為合適步速。訓練應循序漸進，控制每周只增加10%訓練量，距離或速度二擇其一。
場地亦很重要，當中以運動場最為合適，其平坦地面吸震力強，對關節起保護作用。要注意的，是跑手或因應場地規則單方向跑，轉彎時膝關節重複受壓，導致勞損。若場地許何，可以反方向跑相同圈數，平衡膝關節壓力。一般石屎路其實不太適合跑步，建議選擇抗震性較佳的跑鞋，但要注意壽命，有研究指跑畢400至800公里後，抗震功能會大幅下降，所以切勿等跑鞋爛掉才更換。
