蘋果生活：《無格豪情》殺入書展
肥龍出新作解構日AV發展

【本報訊】一眾港男相信對「肥龍」並不陌生，屬90年代豪情夜生活代表人物。電影《3D豪情》最近熱賣，不但勾起港佬集體回憶，更激起「肥龍」出山，直指電影中日本成人業過於漫畫化，決定將當年在日本涉獵風俗事業真實一面，以至親身接觸AV女優的體驗，結集成書，取名《無格豪情》，殺入灣仔書展開賣。
記者：蔡朗清

《無格豪情》出版商紅出版集團總經理林達昌表示，在台灣有不少探討成人事業的文化書籍出版，本港則較為少見，而「肥龍」則是本港的經典夜生活作家，屬港男集體回憶，出版社今次邀得「肥龍」撰文，以親身經歷剖析日本當地風俗事業文化及發展，以至從事AV產業的女優心態。林指出，該書不涉淫褻不雅，圖片亦沒有裸露，所以會在本月灣仔書展銷售，「讓讀者真正了解日本成人業文化」。
作者鄭航接受記者訪問時表示，他是當年報章副刊豪情版作者「肥龍」團隊之一，他的新書結集了他在日本風俗事業的所見所聞，「最近杜汶澤主演賣座電影《3D豪情》勾起港人集體回憶，但內容流於誇張漫畫化，所以決定將自己親身見到的真實一面，寫番出嚟。」

掃描QR Code可睇彩色相

外表胖胖的鄭航表示，書中講述他作為首個直擊當地成人電視台直播節目的香港人，「主持人係風俗娘，即係性工作者，製作好專業，現場工作人員會即時為畫面『打格仔』。」他續稱，書中亦講述不同AV女優個案，及從事成人事業的原因，「女優們並非被迫拍AV，而係自願拍戲。好似有一個女優麻丘珠理，在街頭獲星探發掘，拍AV時露出乳房會感到自豪；其男友亦知佢拍AV亦冇問題。」

《無格豪情》剖析日本成人事業發展，從60、70年代從事風俗娘屬於「醜事」，到了80年代則將性工作者捧為「風俗偶象」，以至90年代的援助交際成風等。鄭航指出，日本風俗產業採取會員制度，不接外國人，即使前往採訪成人夜店亦屢碰壁，原因是怕語言溝通產生誤會爭拗，「日本風俗業發展至今，已接受外國人」。
鄭航表示，書中另一突破是採用了QR Code，雖然插圖如AV女優等屬於黑白照，但讀者透過手機的應用程式對應書上的QR Code ，便可以將插圖變為彩色在手機顯現。
