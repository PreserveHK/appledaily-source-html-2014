PIMCO旗艦基金見紅

【本報綜合報道】去年美國經濟持續好轉，加上聯儲局展開退市計劃，令國債價格大幅下跌，拖累全球最大債券基金PIMCO旗下總回報基金（Total Return Fund）出現十四年來首次錄得負回報。

虧損1.92%　14年來首見

據財經資訊公司Morningstar的數據，PIMCO總回報基金去年為負1.92%，遠遜於2012年的10.4%回報率，並僅跑贏41%同業。該基金現時資產規模達2,440億美元，當中佔37%投資在美國國債，是「債王」格羅斯（Bill Gross）旗下最大型債券基金。
數據顯示，今次是格羅斯總回報基金1999年來首次出現年度虧損。該基金成立於1987年，在2003年之前只曾於1994年和1999年出現過年度虧損，兩次回報率分別為負3.58%和負0.28%。
即將卸任的聯儲局主席伯南克去年5月曾表示，當美國經濟重回正軌時，儲局將減少買債，導致國債價格在去年下半年持續下跌，10年期債息近日已一度升穿3厘。格羅斯在近期向投資者建議購入短債，認為儲局會維持低息直至2016年或以後。
