BIG SPENDER：傳奇車癡
但求死於車塚中

我2009年已經就李松光的古董甲蟲車珍藏而訪問他，當時他已被診斷出甲狀腺、前列腺及大腸發現癌細胞擴散，醫生斷定最多活三年。到今天，他連脊椎也患癌，心臟還加了個起搏器，早前更曾試過因機器停頓，昏迷近十分鐘後蘇醒，醫生對他「五次也死不去」嘖嘖稱奇，李松光一臉童真：「我捨不得愛車，怎會輕易死呀！」
他一生人買車儲車128架，已年屆80高齡、身患五大重症仍堅持自行維修翻新，還向子女立下遺願，指定以愛車作棺材陪葬，作為古董車迷，為心愛之物原來可以去到咁盡。
記者：徐影龍　攝影：伍慶泉

爛地車塚藏歐陸住宅

李松光的車場位處上水新田，外望是一大片荒廢之地，佈滿了待修翻新的爛車，包括Alfa Romeo GTV、Bugatti Type 35、萬事得蓋世太保Cosmo、二戰甲蟲等等，認識古董車之人，才知這裏充滿寶藏，早年曾有古董車會到訪，統計及登記他的愛車收藏量，證實藏品竟達到128架，可惜地方有限，無法整齊地陳列出來，許多都被草草埋在地下。「除了沒時間打理，地方也是一個問題，不是放棄了它們，只是把地底當作停車場而已，明白自己擁有它們便足夠，風采便留在心中吧。」滿臉蓄着白鬍子，衣履整潔的他說道。
中央自蓋的房子也是殘舊不堪，踏入屋內又是另一番景象，全歐陸式古典佈置，放着三張舊而不殘的寬大梳化、都市人夢寐以求的紅磚壁爐、12呎高天花板吊着典雅英式水晶燈飾，四處還擺放了FM膽式收音機、8×10吋古老西德木製相機等，讓人錯覺走進了古歐洲時代，渾忘了現實中正身處爛地車場。

患癌仍不停儲車

年少時一直春風得意，李松光伙拍黃霑創作一系列廣告，名氣跟收入越滾越大，亦曾當過演員、導演及賽車手，到現在儘管五大惡疾纏身，卻依然精神奕奕，每天在上水車場四出巡視心愛收藏品，興之所至還會動手維修引擎、波箱，近年雖已病得無氣無力，難以再搬抬及裝拆大型部件，但仍會四出檢視愛車。拍攝當天我跟着李松光在車場四處逛，當他在Alfa Romeo車底發現了遺失多時Bugatti Type 35的門柄，即高興得開懷大笑手舞足蹈，自認今天行運。每天點算愛車、搜尋遺失零件，已是李松光生活一部份了。

貴車難修　鍾愛平民化

李松光16歲在外國考牌揸電單車，入手第一部便是得吉利90cc，之後心思思玩大型車，便轉揸哈利，儘管是電單車中的名牌，但李松光嫌手動波箱不好揸而放棄，最後愛上了寶馬有陸上之王美譽的R25 250cc。「這款是電影《大逃亡》主角在球場逃亡時的那款，我很喜愛史提夫麥昆，到1985年機緣下才能購入此車，一圓心願。」面對四輪私家車，李松光也是偏愛平民品牌而不愛名牌。「當年勞斯萊斯及積架名氣很大，但古董車零件及維修成本太高，亦曾有車主用法拉利Dino，交換我的古董Benz跑300SL，不過因為Dino不好駕馭而拒絕了。後來因法拉利老闆Enzo Ferrari的兒子Dino死了，車價即時漲升十倍，但我也不後悔當初決定，因金錢不能衡量自己喜愛的收藏。」除了平民車，李松光對開篷車也是情有獨鍾。「一世人也在追求開篷車，很喜歡那無拘無束的感覺，就算下雨中也不會停下關篷（古董開篷車大多需人手裝拆或拉下篷頂），任雨打在身上，讓我很有感覺。70年代由傳奇賽車手潘炳烈代理之Alfa Romeo，找了李松光駕着很前衞的GTV拍宣傳廣告，拍得他瀟灑到不得了，自此李松光也戀上Alfa Romeo了。

專收第一代車款

談到珍藏，他收藏的不是全港唯一一部，便是車廠系列的第一代，如二戰時期的德軍軍車、1952年出廠的甲蟲、第一代Alfa Romeo、寶馬三系開篷車、MG第一代Midget等，認真翻新後，絕對賣到好價錢。但論經典之最，首推Bugatti的經典跑車Type 35，李松光沒說出口，但維修翻新得最多，到目前亦最企理的，便是這台Bugatti。「程小東弟弟程小龍在英國購入回港，輾轉流落在我朋友手上，最後看在友情份上，割愛讓了給我。全港僅得一台，很多部位也是很有魅力，如以皮帶扣着的引擎蓋、手掣裝置在車身外邊等。那張兩座位皮椅，費了很久才找到，物主索價二萬多元，很貴但值得，因為椅上的壓紋圖案實在很美，像為Bugatti添上了靈魂一樣。」
儲車無數，但李松光又因病患而無力翻新古董愛車，未來計劃已沒打算勉強維修下去了。「現在經常出入醫院，沒時間打理珍藏了，唯一心願是死後，以Bugatti Type 35或古董Benz跑作棺材，那便心滿意足了。」聽者心酸，但說者一派樂天從容，彷彿認為能跟愛車葬在一起，是一件比生存更重要的事。
