【Game移動】《The Need for Speed》20年　越開越快

第一款《The Need for Speed》是1994年推出的，轉眼間就已經過了20年。相信玩過第一集《The Need for Speed》的朋友，現在都差不多三字頭了。

回看以前的畫面，其實真的感受不到當年「Speed」的感覺，不過童年回憶永憶是無價的。官方最近發佈了一條20周年紀念短片，剪輯了歷來幾集的《The Need for Speed》遊戲短片，由最舊的《The Need for Speed》到最新的《Need For Speed: Rivals》都有。最精彩是還有《The Need for Speed》電影的片段，看到那些過千萬超跑炒到飛天，真的非常興奮，好想立即落場走一轉！
