港大生搞Dry Club
安撫過萬dry爆大學生

HKU Dry Club，一個在大專界人氣爆燈名字。Dry Club不是港大的一個普通學會、也不是配對公司，其實只是一個普通Facebook專頁。這個零成本的專頁，靠著4個Admin(管理員)不眠不休地經營，短短一年半時間，憑同學間的口耳相傳，影響力悄悄由港大伸延到其他大學，成功俘虜17000個大學生寂寞的心。

早前HKU Dry Club出售印上諷刺字詞的衞衣，瞬間賣出3000件，勁賺六位數，更令港大校園一條長長人龍等買衫，上月情人節他們更在蘭桂坊搞party，助一眾「乾燥」大學生「脫dry」。有人認為4位大學生有創意，但亦有人批評他們太商業化。究竟什麼來是Dry Club？Admin們又是何方神聖呢？《蘋果》訪問兩個死都不肯上鏡、還要求訪問片段要「變聲」的Admin，揭開Dry Club的成功之謎。
記者︰徐麗薇
攝影︰馮就康、王俊龍、陳軍誌

記：記者
Admin #1︰Dry Club創辦人（港大「神科」二年級生）
Admin 魂︰Dry Club 始創Admin（港大「頹科」二年級生）

記︰究竟HKU Dry Club是怎樣誕生的？

Admin #1︰我覺得港大學生的生活很枯燥，上學等放學，放學等放假，令我開始反思大學的價值。因為這樣，有天突發奇想，想辦一個像港大的9gag，給同學們一個平台嬉笑怒罵。

Admin 魂︰當#1有了這個念頭後，想找一個人合作，而他又覺得我是個挺有趣的人，自然就走在一起，促成了Dry Club的誕生。

記︰你們怎麼理解「Dry」，有特別的含義嗎？

Admin #1︰Dry其實泛指很多東西，不只是欠缺愛情的滋潤，也可能是對生活的不滿，也會用dry字來形容。

記︰那Dry Club到底是怎樣的一個page？

Admin 魂︰開始的時候是以匿名的形式，讓同學們可以投稿，我們post出來，主題大多圍繞愛情甚至大學生活，幫同學說出放在心裡不敢說出口的話。

記︰是有點像白姐姐的《盡訴心中情》嗎？

Admin #1︰哈哈，對，我們4個admin會24小時輪流管理專頁，處理同學們的投稿。我們也會發放一些自家「改圖」，能引起話題也好，「得啖笑」也好，這都是我們的特色。

記︰Dry Club為何會成功？

Admin 魂︰我們與其他專頁不同的地方是因為有admin的留言，每個人代表著不同的性格，有時一些留言能一矢中的地說中大學生的心聲，讚好就是這樣得來的。

Admin #1︰對，有時admin的留言得到的like比起本身那個post還要多，但我們沒有刻意隱密身份，其實我們是誰根本無關重要，重要的是我們更夠膽說出一些其他人不敢說的話，如支持同性戀平權，引起大家的共鳴。

記︰有人說你們越來越商業化了，變成一盤生意？

Admin 魂︰確實真的有人找過我們賣廣告，公司也有學會也有機構也有，但我們都盡量避免。

Admin #1︰我們不想Dry Club淪為宣傳易，因為這不是別人like這個page的原因。

記︰但聽說你們準備跟人合作搞啤酒節，這不就是商業化嗎？

Admin 魂︰其實我們真的不是志在賺錢，像情人節前夕的那個party，我們賺得不多，但收到不少回應說那個party真的撮合了不少情侶，一起共渡情人節。能真正的替同學們「脫dry」，這才是令我們高興的事。

最後，經歷過一年半搞Dry Club、辦活動經驗，究竟大學生怎樣才能「脫dry」？Admin 魂表示：「人們經常說女神『收兵』，那為何你要做她的兵？切實地作出改變，令自己成為女神欣賞的人，踏出你的第一步，去跟她表白。」

「脫dry」，原來是要學懂做一個生活得更有色彩的人。
