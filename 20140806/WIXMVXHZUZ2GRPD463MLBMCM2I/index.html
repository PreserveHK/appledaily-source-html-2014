校園房間內關燈聊天　
醫科生索吻被拒　涉胸襲女同學

【本報訊】香港大學醫學院二年級男生，與女同學參加系會通宵活動，於房內擁抱她後要求接吻，他疑被拒後兩度伸手入她的上衣摸臀、腹及乳房，被控兩項非禮罪受審。女生昨日在庭上直認首次被摸後曾向男生說「Feel ok but I don't like it」，又指男生第二度胸襲她後曾說「Such big breast！」
記者：戴國輝

不認罪的被告劉冠宏（19歲）與事主份屬同學，且在同一系會擔任職務。事主昨在東區裁判法院供稱，去年11月1日晚系會在港大李嘉誠醫學院一活動室舉行換屆儀式，由於她與被告須等候至凌晨出席發問環節，兩人遂到毗鄰房間休息。
其間兩人在沒有開燈的房內聊天，被告提出希望與事主「攬攬」，她以為是家庭成員間的擁抱，於是答應，惟被告越抱越緊，又問她有何感覺。

伸手入衫撫摸

事主提醒被告已有女友，被告轉而提出「Just one kiss」，她拒絕，被告道歉後指自己性格衝動。
兩人繼續談天，被告直言被女友拒絕做愛會感憤怒，又指性事是兩性關係很重要的一環。此時，他坐到事主身旁，伸左手入她的上衣來回摸腹，問她感覺如何，她答「Feel ok but I don't like it」。被告將手伸入她的胸圍搓揉乳房，歷時5至10秒，直至有同學入房始停手。
活動結束後事主留在房內休息，至約上午7時瞓醒時發現房內僅得她與被告。被告重施故技擁抱她，撫摸她的臀及乳房，更說：「Such big breast！」

事主沒有說不

被告緊抓事主右手放到陽具位置，讓她隔着牛仔褲「感受」，又要事主為他口交。事主問他為何不向其他留宿女生提出口交要求，被告回應「Good question」，事主隨後離開房間，被告要她保守秘密。她將事件告訴學長，學長問她是否需要胡椒噴霧，她表示不需要，後來報警。
她承認事發前幾日剛失戀，事發時因擔心會遭被告毆打，故沒有說不。辯方指此舉給被告錯誤訊息，以為可跟她有身體接觸。聆訊今續。
案件編號：ESCC2277/14
