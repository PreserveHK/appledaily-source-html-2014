【Game移動】動漫節前率先睇
BFB實體足球卡

【Game移動】一年一度的動漫節又到了！今年其中一大焦點，《Barcode Footballer》實體球員卡，可讓玩家的球隊火速提升實力，並有機會獲得多位極稀7星球員，勢必熱搶！

Barcode Footballer Card（每張$30）

《Barcode Footballer》推出的實體足球卡，足球卡上顯示的球員，可利用遊戲裡的「條碼」功能而讀取得到該位球星。《Barcode Footballer》實體球員卡包含108位珍貴球星，當中包括從未推出過的法國球王✦施丹！

Barcode Footballer Card Box Set ($750)

每盒均有25張Barcode Footballer Card，當中較高機率包含6星、7星球員，而且內含「球王扭蛋機使用券」1張，於動漫節nxTomo攤位遞交「球王扭蛋機使用券」，便有機會參與每天舉行三次的「球王扭蛋」環節或「ACG限定輕量盃」。抽獎時段每天有三節（下午2時、4時及6時），於每節「球王扭蛋」時節前半小時，官方人員會即場抽取4位放入「球王扭蛋機使用券」到抽獎箱的玩家，並通知4位玩家在指定時間前到達，可上台於特製的「球星扭蛋機」中，抽取該時段四位球王的其中一位從未推出的六大7星球王（必定中獎），包括：✦比利、✦馬勒當拿、✦碧根鮑華、✦薛高、✦柏天尼及✦尤西比奧。

Barcode Footballer Card File ($398)

Barcode Footballer Card File製作精美，可放置全套108張《Barcode Footballer》實體足球卡，最吸引的，是會附送從未推出的7星球員波歷克Barcode Footballer Card一張（技能：死角射球 Lv.3、超級頭槌 Lv.3、致命傳球 Lv.3），萬勿錯過。
