TGS2014率先睇　PS4暗黑掛帥　強推《勇者》特別版

相比上年新機年的熱鬧，今年Tokyo Game Show（TGS）激情有所減弱，主場大佬PlayStation除了推出《勇者鬥惡龍》特別版PS4，還發動game海戰術，《Bloodborne》和《The Order: 1886》延續數月前美國E3強勢，繼續做主力。加上《Biohazard Revelations 2》和《Psycho Break》全部打怪獸，可謂四大皇牌暗黑作。官方更宣佈即將進軍中國，大量作品中文化，《人中之龍 0》首次中文化已令機迷興奮。

一如以往，今年TGS在幕張舉行，展期為9月18至21日。PlayStation攤位聲勢浩大，正面擺出《Bloodborne》和《The Order: 1886》雙鬼拍門陣，試玩人數不少。前者玩暴力埋身戰，見血打怪獸；後者口味較輕，武器設計精采萬分。不過兩作都在E3玩過，對記者吸引力不大，反而對《Biohazard Revelations 2》和《Psycho Break》更有興趣。

恐怖打怪　四大作吸人氣

《Biohazard Revelations 2》由前作船上移到孤島，背景更大，玩家還可隨時切換女角Claire和Moira，亦能試打新怪"The Afflicted"。雖沒強大武器，但動作相當快，記者挑戰多次都失敗而回。可是廠方宣佈了一個相信會令機迷講粗口的消息，就是遊戲將於2015年初，分四個星期以四章推出，想一次過買完全版就要等四個星期，認真吊癮！
《Psycho Break》其實是日版《The Evil Within》，同是第三身恐怖射擊，試玩頗有《Biohazard》感覺，遊戲中不乏爆頭環節，玩家打死怪物後更可燒屍，同時亦要小心機關，記者就試過被大齒輪弄死無全屍！不過視點仍有改進空間，打怪時有盲點，影響戰鬥爽快感，希望正式推出時有改善。

搶眼冧女　史萊姆PS4加粉紅 Vita

主機方面，當然不會那麼快有新機現身，但現場展出的全新Light Pink PS Vita和銀色的《Dragon Quest Heros》（勇者鬥惡龍）特別版PS4，仍相當吸引。前者的糖果粉紅色明顯為女生而設，後者全身銀色，除了機身和手掣印上史萊姆圖案，更附有一個全反光銀色的仿液態金屬立體史萊姆，質感造型都不俗。這個立體史萊姆除了裝飾，更可作USB的防塵位，非常別致有誠意，加上這是一代遊戲經典，相信能吸引不少玩家，但售價及推出日期未定。另外，現場更擺出不少度身訂製PS4和PS Vita機身 。據香港工作人員透露，此服務快將登陸香港，大概二百多元就可訂製自己專屬的靚靚機身，打機更開心！

進攻強國　中文版《人中之龍》

自問是《人中之龍》粉絲，攤位正於PS旁，加上官方宣佈《人中之龍0》將會是系列中首次作中文化，當然要看！攤位外五位女優當然吸睛，但可惜原本期待排隊入密室會有試玩，結果排了半小時卻發現只是入場睇片，可幸短片透露了不少劇情，好像桐生一馬與東城會最兇三幹部見面，還有真島吾郎如何變成獨眼的過去等。
製作人表示團隊致力重建1988年神室町和蒼天崛的環境，新作亦會學《人中之龍維新！》與PS Vita連動。相信是為配合進軍中國，除了《人中之龍 0》，PS4和PS Vita同時推出不少中文版新作，包括《Guilty Gear Xrd -Sign-》、《God Eater 2: Rage Burst》、《Gundam Breaker 2》、《GTA V》和《DRIVECLUB》等，不少更同步推出，令人期待。

娘娘小喬首訪TGS　狂玩Project Morpheus

Game girl是每年亮點，但今年香港有代表！原來「娘娘」杜小喬獲SONY邀請參觀TGS。小喬說：「雖然平時少打機，但第一次來TGS仍相當興奮，場面熱鬧之餘，很多攤位佈置和cosplay都相當精采！」雖說少打機，但她就對PS的Project Morpheus非常有興趣，「戴着這眼罩打機很有趣，像進入了另一個世界，自己不懂游水，但可在試玩中與鯊魚大戰，很刺激呀！」說畢兩句又繼續玩。

記者短評

三國戰紀喜回歸　PS Vita聲勢弱

說實話，今年TGS人氣的確不如往年，只有少數大作排長龍，但相對更易試玩。除了大作，現場亦發現經典橫向格鬥作《三國戰紀》，玩法相似，畫面卻大大提升，亦有舊人物以新造型登場，懷舊驚喜並重。相對PS4，PS Vita聲勢較弱，欠吸引大作，試玩人數亦不多，還望會有更多新作支持PS Vita，以免被機迷遺忘。
