基金觀點：ETF追蹤術語你要知

在選擇投資產品時，投資者都會留意有關產品的各項資訊以加強自己對產品的了解。以債券投資者為例，他們會留意債券產品的年期、利息、孶息率及發行商的信貸評級等；股票投資者則會留意公司業績、派息、市盈率及其他分析指標。對於交易所買賣基金（ETF）的投資者來說，這些以追蹤指數為目標的產品亦不乏值得留意的地方。當中有一項可能使不少投資者覺得混淆的，就是追蹤偏離度及追蹤誤差，那麼兩者之間有甚麼分別呢？

誤差低　穩定性較高

追蹤偏離度（Tracking Difference）指於某指定時間內，ETF產品與其追蹤指數的回報差距。值得留意的是，投資者於計算ETF產品的回報時，應使用產品基金的每股資產淨值（Net Asset Value 或 NAV）而不是產品本身的交易價格。例如，於某個交易日，某ETF產品的每股資產淨值由10元上升至10.1元（回報1%），而產品所追蹤的指數由20,000點升至20,220點（回報1.1%），此產品於該日的追蹤偏離度為 -0.1%。
追蹤誤差（Tracking Error）指於某指定時間內，ETF產品追蹤偏離度的波動性（一般使用統計學的標準差來計算）。投資者可以利用追蹤誤差來評估ETF產品於追蹤指數上的穩定性。假設有兩隻結構相同，追蹤同一個指數的ETF。於同一段時間內，產品甲的追蹤誤差為1.5%，產品乙的追蹤誤差為2.0%，因此產品甲於追蹤指數上有較高的穩定性。
