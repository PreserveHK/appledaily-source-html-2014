學黑珍珠美容　化身無添加美魔女

有天媽媽告訴我，「這紙面膜有效期至明年，可想而知內含多少防腐劑！」事實的確如此，所以歐美一早盛行在家鮮榨果汁敷臉，連有黑珍珠之稱的名模Naomi Campbell亦以此天然方法為肌膚保鮮。要化身美魔女，原來方便又簡單。
記者：許 政
攝影：陳永威

“坊間面膜成份　不能媲美鮮榨果汁”

國際專業芳香治療師協會註冊香薰芳香治療師培訓導師何嘉瑩(Alice)指出，坊間由數十元至數百元的美顏面膜，大都含大量化學色素和香料，有機會灼傷皮膚之餘，更可能致癌。Alice 說自己從小皮膚是粗糙的暗瘡性皮膚，曾為坊間多個品牌面膜的「信徒」，月擲萬元購買護膚品，可惜膚質未見起色。今天的她皮膚瑕疵不再，透紅面色健康有光澤。她說以果汁自製面膜是入門，「水果的營養藏於種子和果肉中，惟坊間面膜內的維他命B、C和E等多經人工處理，效果自然較差。」Alice還提醒大家，由於自製面膜無添加防腐劑，建議即製即用為佳。

Mask Type 1：果汁 + 乳酪 = 美白零毛孔肌

除了飲用，乳酪也是絕佳的天然美容聖品。埃及妖后Cleopatra最愛的牛奶浴，其實就是把乳酪加水稀釋，利用牛奶酸化後產生的乳酸精華來護膚。百年德國醫學雜誌Archives of Dermatological Research曾提出，乳酪內含豐富乳酸，有效去角質、美白、減退脂肪酸、黑色素及暗瘡印。加上乳酪性質溫和，配合抗氧能力高的士多啤梨、奇異果和葡萄，或香蕉和牛油果等黏性水果等混合使用均能達到不錯的美容效果。

Alice's Tips：

1. 市面上有多款乳酪選擇，當中以無糖乳酪方可做到最天然無添加的效果。
2. 水果和乳酪以1:1比例較好。因若使用過多維他命C，皮膚可能因不能完全吸收而出現泛黃。
3. 美白果檸檬、青檸及橙均帶反黑成份，於太陽下會讓用家膚色變黑，故不建議使用。

材料：

新鮮士多啤梨
無糖乳酪
蜜糖

Mask Type 2：果汁 + 燕麥片= 舒緩潔淨肌

燕麥片適合油性、乾性和敏感肌人士使用。除了能淨化肌膚雜質，燕麥片特有的葡聚醣能鎖住皮膚水份、有助敏感肌降紅及軟化角質層。不過由於燕麥片需要水份才能溶化成水溶性纖維讓皮膚吸收，故適合配搭多汁蔬果，例如番茄、藍莓、青瓜和紅桑子等。
現年59歲的米雪曾表示，愛以蛋白加麥皮粉自製面膜，再把麥皮粉當作磨砂按摩去除角質層，洗淨後照常做一般護膚程序便可。連50歲的劉美君亦曾表示，平日只用麥皮加水洗面，以保持皮膚彈性。

Alice's Tips：

1. 若面膜太稀，加入麥皮可增加面膜張力，令面膜黏身不易跌下。
2. 把燕麥片打碎，讓磨砂效果更細膩。

材料：

新鮮藍莓
燕麥片
蜜糖

Mask Type 3：果汁 + 蛋黃醬 = 滋潤水嫩肌

蛋黃醬含豐富蛋白質和多種天然油脂，塗上臉感覺像雪糕，可滋潤乾性及老化皮膚；抹在頭髮上可令秀髮變柔亮。蛋黃醬的油份已具保濕效果，故不用加入蜜糖，亦應避免配搭黏性水果，例如香蕉、牛油果、芒果和黑棗。

Alice's Tips：

1. 油性及暗瘡性皮膚不適用。
2. 若想保濕肌膚，可用蛋黃醬混合蛋黃、醋或檸檬，自製護膚液。

材料：

青瓜
蛋黃醬

建議敷臉蔬果

Top Model教路

黑珍珠Naomi Campbell的秘方：
把1/4個牛油果、半個香蕉及一茶匙純酸奶攪成糊狀，敷在潤濕的面部按摩以促進血液循環。若想效果更佳，可按摩後把紗布敷在面膜上，停留10分鐘後洗淨，再薄敷保濕霜。每星期一次。
